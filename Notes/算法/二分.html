二分 ​

​　　二分的整体思想就是通过区间中间值对应的结果来判断是否符合要求，并根据结果来调整区间范围及长度，从而来快速计算出答案或者是减小时间复杂度至O(log2(n))，一般情况下区间的中点mid就是我们所求的答案。
​　　二分的情况有两种分别是浮点二分和整数二分，也对应了两个模板，只要能根据题目确定好初始的区间范围，以及判断条件，套用模板就可以了。

1.浮点二分 ​

python
L="区间左端点"
R="区间右端点"
while L<R:
    mid=(L+R)//2
    if check(mid):
        #如果满足要求，则区间就变成了[mid,R]
        L=mid
    else:
        #如果不满足要求，则区间就要变成[L,mid]
        R=mid

2.整数二分 ​

题目所求答案（一般为整数）具有单调性质
 1、确定初始范围[left,right]
 2、当left≤ right时：
   mid =(left+right)//2
   check(mid)：判断mid是否合法：（定义check函数，什么时候合法，根据题目来确定）
    如果合法：更新ans=mid
    调整左右区间，调整策略为二选一：left=mid+1，right =mid -1【具体题目具体分析】

注意：一般来说，如果题目求的是最小值，那么我们所需要更改的是右区间端点，因为我们要去发现它的左边是否有更小的值满足要求。类似的，对于最大值，我们需要修改的是它的左区间

#整数二分【二分答案】
def check(mid):  #该函数内容的编写根据题目要求来实现
    #判断mid是否满足条件,满足则返回True,否则返回False
    pass
left,right,ans=初始化
while left<=right:
    mid=(left+right)//2  #注意这里求的是整数
    if check(mid):
        ans=mid  #只有合法才会更新答案
        left=mid+1 #二选一
    else:
        right=mid-1 #二选一

题目 ​

[蓝桥杯 2017 省 AB] 分巧克力 ​

题目描述 ​

儿童节那天有 K 位小朋友到小明家做客。小明拿出了珍藏的巧克力招待小朋友们。 小明一共有 N 块巧克力，其中第 i 块是 Hi×Wi 的方格组成的长方形。 为了公平起见，小明需要从这 N 块巧克力中切出 K 块巧克力分给小朋友们。切出的巧克力需要满足：

  1. 形状是正方形，边长是整数。
  2. 大小相同。 例如一块 6×5 的巧克力可以切出 6 块 2×2 的巧克力或者 2 块 3×3 的巧克力。 当然小朋友们都希望得到的巧克力尽可能大，你能帮小 Hi 计算出最大的边长是多少么？

输入格式 ​

第一行包含两个整数 N 和 K。(1≤N,K≤105)。 以下 N 行每行包含两个整数 Hi 和 Wi。(1≤Hi,Wi≤105)。 输入保证每位小朋友至少能获得一块 1×1 的巧克力。

输出格式 ​

输出切出的正方形巧克力最大可能的边长。

样例 #1 ​

样例输入 #1 ​
2 10  
6 5  
5 6
样例输出 #1 ​
2

求解【个人原创】 ​

python
H=[] #存放巧克力的长度
W=[] #存放巧克力的宽度
N,K=map(int,input().split()) #存放巧克力的块数和人数
for _ in range(N):   
    h,w=map(int,input().split())
    H.append(h)
    W.append(w)
L=0
R=max(max(H),max(W))
mid=0
ans=0
def check(x):  #判断函数，用于判断边长为x的巧克力是否满足要求，即每个小朋友都能分到
    sum=0
    for i in range(N):
        sum+=(H[i]//x)*(W[i]//x)  #计算对应一个大块巧克力可以分成多少块指定边长的正方形巧克力
    if sum>=K:
        return True
    else:
        return False

while L<=R:
    mid=(L+R)//2
    if check(mid):
        ans=mid  #符合要求便更新答案
        L=mid+1
    else:
        R=mid-1

print(ans) #输出最终结果
