前缀和 ​

1.一维前缀和 ​

1.概念：前缀和其实就是将一个序列从[0,i]的区间上的所有元素求和对于一个长度为n的序列A，它的前缀和为：sum[i]=A[0]+A[1]+...+A[i]

2.性质:

  • sum[i]=sum[i-1]+a[i]
  • a[l]+...+a[r]=sum[r]-sum[l-1] 【常用于快速求解区间和】

​ 第一条性质通常用于处理出前缀和

​ 第二条性质可以在O(1)的时间内求出区间和

3.实现前缀和

这里会介绍两种实现前缀和的方法：
  • 普通的代码实现：
python
#求出a的前缀和
def presum(A):
    n=len(A)
    sum=[0]*n
    sum[0]=A[0]
    for i in range(1,n):
        sum[i]=sum[i-1]+A[i]
    return sum
def get_sum(sum,l,r):
    if l==0:
        return sum[r]
    else:
        return sum[r]-sum[l-1]
A=list(map(int,input().split()))
l,r=map(int,input().split())
sum=presum(A)
print("前缀和：",sum)
ans=get_sum(sum,l,r)
print(f"区间{l}~{r}的和为：{ans}")

使用迭代工具itertools中的accumulate函数来快速求解序列A的前缀和：

python
from itertools import accumulate
#求出a的前缀和
def presum(A):
    sum=list(accumulate(A))
    return sum
def get_sum(sum,l,r):
    if l==0:
        return sum[r]
    else:
        return sum[r]-sum[l-1]
A=list(map(int,input().split()))
l,r=map(int,input().split())
sum=presum(A)
print("前缀和：",sum)
ans=get_sum(sum,l,r)
print(f"区间{l}~{r}的和为：{ans}")

注意：accumulate只能从序列下标0开始，如果题目是从1开始就需要自行做出调整。

a.例题3419-----小郑的蓝桥平衡串 ​

问题描述

平衡串指的是一个字符串，其中包含两种不同字符，并且这两种字符的数量相等。

例如，ababab 和 aababb 都是平衡串，因为每种字符各有三个，而 abaab和 aaaab都不是平衡串，因为它们的字符数量不相等。

平衡串在密码学和计算机科学中具有重要应用，比如可以用于构造哈希函数或者解决一些数学问题。

小郑拿到一个只包含 L、Q的字符串，他的任务就是找到最长平衡串，且满足平衡串的要求，即保证子串中 L、Q的数量相等。

输入格式

输入一行字符串，保证字符串中只包含字符 L、Q。

输出格式

输出一个整数，为输入字符串中最长平衡串的长度。

样例输入

text
LQLL

样例输出

text
2

题目分析

​ 通过分析题目，我们可以看出平衡串指的是两种字符的数量相同，那么我们将两个字符转换成一对相反数，当数量相同时，这些数的前缀和就为0，那么我们就可以用和为0来作为平衡串的标志，再通过max()来求解出最大的区间长度

解答

python
from itertools import accumulate
def presum(a):
    sum=list(accumulate(a))
    return sum

def get_sum(sum,l,r):
    if l==0:
        return sum[r]
    else:
        return sum[r]-sum[l-1]
words=input()
nums=[]
ans=0
#将对应的'L'和'Q'转换成1和-1
for word in words:
    if word=='L':
        nums.append(1)
    else:
        nums.append(-1)
#求解前缀和
sum=presum(nums)
n=len(sum)
#求出前缀和数组中，和为0的最大区间长度
for i in range(n):
    for j in range(i,n):
        if get_sum(sum,i,j)==0:
            ans=max(ans,j-i+1)
print(ans)
b.例题3829-----大石头的搬运工 ​

问题描述

在一款名为”大石头的搬运工“的游戏中，玩家需要操作一排 n堆石头，进行 n−1 轮游戏。

每一轮，玩家可以选择一堆石头，并将其移动到任意位置。

在 n−1轮移动结束时，要求将所有的石头移动到一起（即所有石头的位置相同）即为成功。

移动的费用为石头的重量乘以移动的距离。例如，如果一堆重量为 2 的石头从位置 3 移动到位置 5，那么费用为 2×(5−3)=4。

请计算出所有合法方案中，将所有石头移动到一起的最小费用。

可能有多堆石头在同一个位置上，但是一轮只能选择移动其中一堆。

输入格式

第一行一个整数 n，表示石头的数量。

接下来 n行，每行两个整数 w~i~和 p~i~，分别表示第 i堆石头的重量和初始位置。

输出格式

输出一个整数，表示最小的总移动费用。

样例输入

3
2 3
3 1
1 5

样例输出

8

解答

python
n=int(input())
wp=[] #存石头的质量和位置
for _ in range(n):
    wp.append(list(map(int,input().split())))
wp.sort(key=lambda x:x[1]) #按照位置从小到大排序
pre=[0]*n #存放第i+1个石头前的石头全部挪到该位置所付出的价值
pre_weight=wp[0][0] #初始化第一个石头的质量，因为第一个石头前面没有石头要移动，所以付出的价值为0
back=[0]*n#存放第i个石头后的石头全部挪到该位置所付出的价值
back_weight=wp[-1][0] #初始化最后一个石头的质量，因为最后一个石头后面没有石头要移动，所以付出的价值为0

#利用前缀和的知识开始计算移动到对应位置所需要的价值
#计算第a个石头前的
for i in range(1,n):
    pre[i]=pre[i-1]+(wp[i][1]-wp[i-1][1])*pre_weight
    pre_weight+=wp[i][0]
#计算第a个石头后的
for i in range(n-2,-1,-1):
    back[i]=back[i+1]+(wp[i+1][1]-wp[i][1])*back_weight
    back_weight+=wp[i][0]

ans=pre[0]+back[0]
for i in range(n):
    result=pre[i]+back[i]
    if result < ans:
        ans=result
print(ans)

2.二维前缀和 ​

下标从(1,1)开始，有一个统一的公式 【切记，小标是从1开始并不是从0开始】

1.如何计算前缀和矩阵：

S[x][y]指的是点（x，y）左上角的所有数之和

S[x][y]=S[x][y-1]+S[x-1][y]-S[x-1][y-1]+num[x][y] 【num[x][y]为（x，y）点的数】

2.如何利用前缀和矩阵，计算某一个子矩阵的和

​ 子矩阵[x1,y1,x2,y2]的和：S[x2][y2]-S[x2][y1-1]-S[x1-1][y2]+S[x1-1][y1-1]

模板代码：

  • 如何求解二维前缀和
python
#输出二维矩阵
def output(num,n):
    for i in range(1,n+1):
        print(' '.join(map(str,num[i][1:])))
#读取矩阵的行数和列数
n,m=map(int,input().split())        
nums=[[0]*(m+1) for _ in range(n+1)]
sums=[[0]*(m+1) for _ in range(n+1)]
#输入一个二维数组
for i in range(1,n+1):
    nums[i]=[0]+list(map(int,input().split()))
output(nums,n)
#计算二维数组的前缀和
for i in range(1,n+1):
    for j in range(1,m+1):
        sums[i][j]=sums[i-1][j]+sums[i][j-1]-sums[i-1][j-1]+nums[i][j]
output(sums,n)
  • 如何求解子矩阵的和
python
print("输入子矩阵区间：")
x1,y1,x2,y2=map(int,input().split())
ans=sums[x2][y2]-sums[x1-1][y2]-sums[x2][y1-1]+sums[x1-1][y1-1]
print(ans)
